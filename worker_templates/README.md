# תבניות אקסל לעובד

תיקייה זו מכילה תבניות `.xlsx` לעובד שמכין את הנתונים הראשוניים למערכת.

## הקבצים

| קובץ | מה למלא |
|---|---|
| `employees.xlsx` | רשימת כל העובדים: מספר עובד, שם, טלפון, מקצוע, הרשאה |
| `employee_availability.xlsx` | לוח זמינות לתקופה 1.5.2026 — 12.7.2026 (V/X לכל יום) |
| `vehicles.xlsx` | רשימת הרכבים/ציוד: מספר רכב, מקצוע, מחלקה, תת מחלקה |

## הוראות

לכל קובץ יש שני גיליונות:
- **`data`** — שם העובד ממלא את הנתונים. בקובץ קיימות שורות דוגמה באפור — מחק אותן והכנס את הנתונים האמיתיים.
- **`הוראות`** — הסבר מפורט בעברית על המילוי.

## שליחה לעובד

ניתן לשלוח את שלושת הקבצים בנפרד או יחד. לאחר ההחזרה, הקבצים יובאו אל המערכת אוטומטית.

## הקובץ שלא נמצא כאן

`parts.xlsx` (קטלוג חלקי החילוף) **יגיע מהעובד** עם הסכמה שהוא מגדיר. לאחר שיגיע, נטמיע את הקטלוג במערכת.

## חידוש

אם תרצה לחדש את התבניות (למשל אחרי שינוי במסמך הדרישות):

```bash
python3 scripts/generate_excel_templates.py
```
